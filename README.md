# Git Legacy

Git Legacy is a port of Git and its most commonly used plugins to older releases of Windows.

| ------- | ------ | ----- |
| Windows 8 | ⏳ Early work in progress |
| Windows 7 | ⏳ Early work in progress |

| Windows XP | ❌ Not in active development |
